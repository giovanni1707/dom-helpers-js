DOM Helpers Js

A powerful, high-performance vanilla JavaScript DOM manipulation library

Transform how you interact with the DOM using intelligent caching, automatic enhancement, and a declarative API that makes complex operations simple.

Get Started View Docs GitHub
npm install @giovanni1707/dom-helpers

Why DOM Helpers?

Built for developers who love vanilla JavaScript but want better tools

Intelligent Caching

Smart caching system ensures lightning-fast element retrieval without redundant DOM queries.

Declarative API

Update multiple properties, styles, and listeners with a single method call. Clean and intuitive.

Zero Dependencies

Pure vanilla JavaScript with no external dependencies. Lightweight and reliable.

Fine-Grained Updates

Intelligent system updates only what changed, minimizing DOM writes and maximizing performance.

Modular Architecture

Use only what you need. Core library plus optional modules for storage, forms, animations, and more.

Browser Compatible

Works across all modern browsers with support back to IE 9+. Production-ready.

Compare vanilla JavaScript with DOM Helpers

Traditional JavaScript

Verbose & Repetitive
script.js
const myButton = document.getElementById('myButton');
myButton.textContent = 'Enhanced Button!';
myButton.style.backgroundColor = '#007bff';
myButton.style.color = 'white';
myButton.style.padding = '10px 20px';
myButton.addEventListener('click', () => {
    alert('Button clicked!');
});

const messages = document.getElementsByClassName('message');
for (let msg of messages) {
    msg.style.color = 'blue';
    msg.style.fontWeight = 'bold';
}

const container = document.querySelector('.container');
container.style.border = '2px solid #ccc';
container.style.padding = '20px';

DOM Helpers

Clean & Declarative
script.js
Elements.myButton.update({
    textContent: 'Enhanced Button!',
    style: {
        backgroundColor: '#007bff',
        color: 'white',
        padding: '10px 20px'
    },
    addEventListener: {
        click: () => alert('Button clicked!')
    }
});

Collections.ClassName.message.update({
    style: { color: 'blue', fontWeight: 'bold' }
});

Selector.query('.container').update({
    style: { border: '2px solid #ccc', padding: '20px' }
});
