Fine-Grained Control System

NEW! Advanced Performance System

DOM Helpers includes an intelligent fine-grained control system that minimizes DOM writes and maximizes performance across all helper modules.

What is Fine-Grained Control?

Fine-grained control means DOM Helpers intelligently compares new values with previously applied values before making any DOM changes. This prevents unnecessary reflows, repaints, and duplicate operations.

Smart Property Comparison

// Only updates if value actually changed
Elements.myButton.update({
  textContent: "Click Me" // ✅ Updates DOM
});

Elements.myButton.update({
  textContent: "Click Me" // ✅ Skips - already set
});

Granular Style Updates

// Updates only changed CSS properties
Elements.myDiv.update({
  style: { color: "blue", fontSize: "16px" }
});

Elements.myDiv.update({
  style: { color: "blue", padding: "10px" }
  // color skipped, padding updated
});

Event Listener Deduplication

function handleClick(e) {
  console.log('Clicked!');
}

// First call - adds listener
Elements.myButton.update({
  addEventListener: ['click', handleClick]
});

// Second call - skips (same reference)
Elements.myButton.update({
  addEventListener: ['click', handleClick]
});

Deep Object Comparison

// Compares object/array contents
Elements.myElement.update({
  dataset: { userId: "123", role: "admin" }
});

Elements.myElement.update({
  dataset: { userId: "123", role: "admin" }
  // ✅ Skips - identical
});

Performance Benefits

Operation Without Fine-Grained With Fine-Grained Improvement
Repeated textContent Every update writes to DOM Only first write ~95% reduction
Style updates Overwrites all properties Updates only changed ~70% reduction
Event listeners Multiple listeners added Deduped automatically 100% prevention
Attribute updates Always sets attributes Only when different ~80% reduction

Memory Management

Fine-grained tracking uses WeakMaps for automatic garbage collection:

// ✅ Automatic cleanup when elements are removed
const element = document.createElement('div');
element.update({ textContent: "Test" }); // Tracked in WeakMap

document.body.appendChild(element);
document.body.removeChild(element); // WeakMap entry auto-removed

Elements Helper

The Elements helper provides instant access to any element by its ID, with intelligent caching for maximum performance and automatic enhancement with the universal .update() method.

Basic Usage

// Access element by ID - automatically cached and enhanced
const button = Elements.myButton;
const form = Elements.userForm;
const modal = Elements.confirmModal;

// Check existence before use
if (Elements.myButton) {
    console.log('Button found!');
}

// Get multiple elements with destructuring
const { header, footer, sidebar } = Elements.destructure('header', 'footer', 'sidebar');

// Wait for dynamic elements
const dynamicButton = await Elements.waitFor('dynamicButton');
const dynamicElements = await Elements.waitFor('btn1', 'btn2', 'btn3');

Enhanced Update Method

// Comprehensive element update - all in one call
Elements.myButton.update({
    // Basic properties
    textContent: 'Enhanced Button',
    innerHTML: 'Click Me',
    id: 'newId',
    className: 'btn btn-primary',
    
    // Style object - batch CSS updates
    style: {
        backgroundColor: '#007bff',
        color: 'white',
        padding: '10px 20px',
        border: 'none',
        borderRadius: '5px',
        cursor: 'pointer',
        transition: 'all 0.3s ease'
    },
    
    // classList methods - enhanced with arrays
    classList: {
        add: ['active', 'highlighted', 'enhanced'],
        remove: ['disabled', 'hidden'],
        toggle: 'pressed',
        replace: ['old-theme', 'new-theme']
    },
    
    // Attributes - multiple formats supported
    setAttribute: {
        'data-role': 'button',
        'aria-label': 'Enhanced button',
        'title': 'Click to perform action',
        'data-user-id': '123'
    },
    
    // Remove attributes
    removeAttribute: ['disabled', 'readonly'],
    
    // Dataset - direct data attribute access
    dataset: {
        userId: '123',
        action: 'submit',
        category: 'primary',
        timestamp: Date.now()
    },
    
    // Event listeners - multiple events at once
    addEventListener: {
        click: (e) => {
            console.log('Button clicked!', e);
            e.target.update({
                classList: { toggle: 'clicked' },
                style: { transform: 'scale(0.95)' }
            });
        },
        mouseenter: (e) => e.target.update({ style: { opacity: '0.8' } }),
        mouseleave: (e) => e.target.update({ style: { opacity: '1' } }),
        focus: [handleFocus, { passive: true }] // With options
    },
    
    // Method calls - invoke methods with arguments
    focus: [],
    scrollIntoView: [{ behavior: 'smooth', block: 'center' }],
    click: []
});

Key Features

  • Automatic Caching: Elements are cached after first access for O(1) retrieval. Subsequent accesses return the cached element instantly.
  • Enhanced Methods: Every element automatically gets an enhanced .update() method for declarative DOM manipulation.
  • Error Resilient: Gracefully handles missing elements without throwing errors. Returns undefined for non-existent elements.
  • Proxy-Based Access: Uses JavaScript Proxy for seamless ID-based element access without method calls.
  • Fine-Grained Updates: Only updates DOM properties that have actually changed, preventing unnecessary reflows.

Safe Access Pattern

// ✅ Good - Always check existence
if (Elements.myButton) {
    Elements.myButton.update({ textContent: 'Found!' });
}

// ✅ Good - Use safe access with default
const button = Elements.get('maybeButton', null);
if (button) {
    button.update({ textContent: 'Exists!' });
}

Interactive Navigation Menu

function createInteractiveNavigation() {
    const nav = {
        menuButton: Elements.menuButton,
        navMenu: Elements.navMenu,
        navItems: Collections.ClassName.navItem,
        isOpen: false
    };
    
    // Setup menu toggle
    nav.menuButton.update({
        addEventListener: {
            click: (e) => {
                e.preventDefault();
                nav.isOpen = !nav.isOpen;
                
                nav.navMenu.update({
                    classList: { toggle: 'open' },
                    style: { display: nav.isOpen ? 'block' : 'none' }
                });
                
                nav.menuButton.update({
                    textContent: nav.isOpen ? '✕' : '☰',
                    setAttribute: { 'aria-expanded': nav.isOpen.toString() }
                });
            }
        }
    });
    
    // Setup navigation items
    nav.navItems.update({
        addEventListener: {
            click: (e) => {
                // Remove active from all, add to clicked
                nav.navItems.update({ classList: { remove: 'active' } });
                e.target.update({ classList: { add: 'active' } });
            },
            mouseenter: (e) => {
                e.target.update({ style: { backgroundColor: '#f0f0f0' } });
            },
            mouseleave: (e) => {
                e.target.update({ style: { backgroundColor: '' } });
            }
        }
    });
    
    // Responsive behavior
    window.addEventListener('resize', () => {
        if (window.innerWidth > 768 && nav.isOpen) {
            nav.isOpen = false;
            nav.navMenu.update({ style: { display: '' } });
        }
    });
    
    return nav;
}

Dynamic Content Loader

function createContentLoader() {
    const loader = {
        container: Elements.contentContainer,
        loadingIndicator: Elements.loadingIndicator,
        errorContainer: Elements.errorContainer,
        isLoading: false
    };
    
    loader.loadContent = async function(url, options = {}) {
        if (this.isLoading) return;
        
        // Set loading state
        this.isLoading = true;
        this.loadingIndicator?.update({ style: { display: 'block' } });
        this.container?.update({ classList: { add: 'loading' } });
        this.errorContainer?.update({ style: { display: 'none' } });
        
        try {
            const response = await fetch(url, {
                method: options.method || 'GET',
                headers: options.headers || {},
                body: options.body
            });
            
            if (!response.ok) {
                throw new Error(`HTTP error! status: ${response.status}`);
            }
            
            const content = await response.text();
            
            // Display content with fade-in
            this.container.update({
                innerHTML: content,
                style: { opacity: '0', transition: 'opacity 0.3s ease' }
            });
            
            setTimeout(() => {
                this.container.update({ style: { opacity: '1' } });
            }, 50);
            
            // Enhance new content
            Selector.queryAll('#contentContainer .btn').update({
                style: { transition: 'all 0.3s ease', cursor: 'pointer' },
                addEventListener: {
                    mouseenter: (e) => {
                        e.target.update({ style: { transform: 'translateY(-2px)' } });
                    },
                    mouseleave: (e) => {
                        e.target.update({ style: { transform: 'translateY(0)' } });
                    }
                }
            });
            
        } catch (error) {
            this.errorContainer?.update({
                textContent: `Error loading content: ${error.message}`,
                style: {
                    display: 'block',
                    color: '#dc3545',
                    padding: '20px',
                    backgroundColor: '#f8d7da',
                    border: '1px solid #f5c6cb',
                    borderRadius: '4px'
                }
            });
        } finally {
            this.isLoading = false;
            this.loadingIndicator?.update({ style: { display: 'none' } });
            this.container?.update({ classList: { remove: 'loading' } });
        }
    };
    
    // Setup navigation
    Selector.queryAll('[data-load-content]').update({
        addEventListener: {
            click: (e) => {
                e.preventDefault();
                const url = e.target.getAttribute('data-load-content');
                if (url) {
                    loader.loadContent(url);
                    Selector.queryAll('[data-load-content]').update({
                        classList: { remove: 'active' }
                    });
                    e.target.update({ classList: { add: 'active' } });
                }
            }
        }
    });
    
    return loader;
}

// Usage
const contentLoader = createContentLoader();
contentLoader.loadContent('/api/dashboard');

function createDynamicButtonManager() {
    const manager = {
        container: Elements.container,
        addButton: Elements.addButton,
        counter: Elements.counter,
        buttonCount: 0,
        clickCount: 0
    };
    
    // Event delegation
    manager.container.update({
        addEventListener: {
            click: (e) => {
                if (e.target.matches('.dynamic-btn')) {
                    e.target.update({
                        style: { backgroundColor: '#28a745', transform: 'scale(0.95)' },
                        textContent: 'Clicked!'
                    });
                    
                    setTimeout(() => {
                        e.target.update({ style: { transform: 'scale(1)' } });
                    }, 150);
                    
                    manager.counter.update({
                        textContent: `Buttons clicked: ${++manager.clickCount}`
                    });
                }
                
                if (e.target.matches('.remove-btn')) {
                    const buttonItem = e.target.closest('.button-item');
                    buttonItem.update({
                        style: { opacity: '0', transform: 'translateX(-100%)' }
                    });
                    
                    setTimeout(() => {
                        buttonItem.remove();
                        Collections.ClassName.dynamicBtn.forEach((btn, index) => {
                            btn.update({ textContent: `Button ${index + 1}` });
                        });
                        manager.buttonCount = Collections.ClassName.dynamicBtn.length;
                    }, 300);
                }
            }
        }
    });
    
    // Add button functionality
    manager.addButton.update({
        addEventListener: {
            click: () => {
                const buttonItem = document.createElement('div');
                buttonItem.update({
                    className: 'button-item',
                    style: {
                        opacity: '0',
                        transform: 'translateX(100%)',
                        transition: 'all 0.3s ease'
                    }
                });
                
                const button = document.createElement('button');
                button.update({
                    className: 'dynamic-btn',
                    textContent: `Button ${++manager.buttonCount}`,
                    style: {
                        padding: '8px 16px',
                        margin: '5px',
                        border: 'none',
                        borderRadius: '4px',
                        backgroundColor: '#007bff',
                        color: 'white',
                        cursor: 'pointer'
                    }
                });
                
                const removeBtn = document.createElement('button');
                removeBtn.update({
                    className: 'remove-btn',
                    textContent: '×',
                    style: {
                        marginLeft: '10px',
                        padding: '4px 8px',
                        border: 'none',
                        borderRadius: '4px',
                        backgroundColor: '#dc3545',
                        color: 'white',
                        cursor: 'pointer'
                    }
                });
                
                buttonItem.appendChild(button);
                buttonItem.appendChild(removeBtn);
                manager.container.appendChild(buttonItem);
                
                setTimeout(() => {
                    buttonItem.update({
                        style: { opacity: '1', transform: 'translateX(0)' }
                    });
                }, 50);
            }
        }
    });
    
    // Initialize counter
    manager.counter.update({ textContent: 'Buttons clicked: 0' });
    
    return manager;
}

Best Practices

Element Access

✅ Good - Use appropriate helper

const btn = Elements.myButton;
const btns = Collections.ClassName.btn;
const form = Selector.query('#form');

✅ Good - Check existence

if (Elements.myButton) {
  Elements.myButton.update({...});
}

Common Pitfalls

❌ Avoid - Multiple separate updates

Elements.btn.textContent = 'Text';
Elements.btn.style.color = 'blue';
Elements.btn.classList.add('active');

❌ Avoid - No existence check

Elements.missingBtn.update({...});
// Could fail silently

Update Method Best Practices

✅ DO: Batch Multiple Updates

// ✅ Efficient - Single update call
Elements.myElement.update({
  textContent: 'New content',
  style: { color: 'blue', fontSize: '16px' },
  classList: { add: 'active' },
  addEventListener: { click: handleClick }
});

✅ DO: Use Function References for Events

// ✅ Good - Reusable function reference
const handleClick = (e) => console.log('Clicked!');

Elements.myButton.update({
  addEventListener: { click: handleClick }
});

✅ DO: Cache Frequently Accessed Elements

// ✅ Efficient - Cache elements
const ui = {
  header: Elements.header,
  nav: Elements.navigation,
  content: Elements.mainContent,
  footer: Elements.footer
};

// Use cached references
ui.header.update({ style: { position: 'fixed' } });

Choose the Right Helper

⚡
Fastest: Elements (ID access)
Elements.myButton
⚡
Fast: Collections (native APIs)
Collections.ClassName.btn
⚡
Flexible: Selector (with caching)

// ✅ Efficient - Single update
Collections.ClassName.btn.update({

Collections.ClassName.btn.forEach(btn => {

Extending Functionality

// Add custom methods to all elements
function enhanceWithCustomMethods() {
    const originalEnhance = ProductionElementsHelper.prototype._enhanceElementWithUpdate;
    
    ProductionElementsHelper.prototype._enhanceElementWithUpdate = function(element) {
        element = originalEnhance.call(this, element);
        
        // Add custom fadeIn method
        element.fadeIn = function(duration = 300) {
            this.style.opacity = '0';
            this.style.transition = `opacity ${duration}ms ease`;
            this.style.display = 'block';
            
            setTimeout(() => {
                this.style.opacity = '1';
            }, 10);
            
            return this;
        };
        
        // Add custom fadeOut method
        element.fadeOut = function(duration = 300) {
            this.style.transition = `opacity ${duration}ms ease`;
            this.style.opacity = '0';
            
            setTimeout(() => {
                this.style.display = 'none';
            }, duration);
            
            return this;
        };
        
        return element;
    };
}

// Apply enhancements

// Periodic cleanup for long-running applications

  const stats = DOMHelpers.getStats();
  const totalCacheSize = Object.values(stats).reduce(
    (sum, stat) => sum + (stat.cacheSize || 0), 0
