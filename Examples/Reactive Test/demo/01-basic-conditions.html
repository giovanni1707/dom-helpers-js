Understanding Condition Matching

Conditions.js supports multiple types of condition matching patterns. Explore each type below with interactive examples.

1. Boolean Conditions

Match true, false, truthy, or falsy values.
Click a button to test
Conditions.apply(value, { 'true': { textContent: '‚úì Value is true', style: { color: 'green' } }, 'false': { textContent: '‚úó Value is false', style: { color: 'red' } }, 'truthy': { textContent: 'Value is truthy', style: { color: 'blue' } }, 'falsy': { textContent: 'Value is falsy', style: { color: 'gray' } } }, '#boolean-output');

2. String Matching

Match exact strings, check if string includes/starts with/ends with specific text.
Enter text and click test
Conditions.apply(value, { '"hello world"': { textContent: 'Exact match: hello world' }, 'includes:world': { textContent: 'Contains "world"' }, 'startsWith:hello': { textContent: 'Starts with "hello"' }, 'endsWith:world': { textContent: 'Ends with "world"' } }, '#string-output');

3. Numeric Comparisons

Compare numbers using exact values, ranges, or comparison operators (>, <, >=, <=).
Adjust number and click test
Conditions.apply(value, { '0': { textContent: 'Zero' }, '>100': { textContent: 'Greater than 100', style: { color: 'green' } }, '50-100': { textContent: 'Between 50 and 100', style: { color: 'blue' } }, '<50': { textContent: 'Less than 50', style: { color: 'orange' } }, '<0': { textContent: 'Negative', style: { color: 'red' } } }, '#number-output');

4. Regular Expression Matching

Use regex patterns for advanced string matching. Patterns start with / and can include flags.
Try entering an email address
Conditions.apply(value, { '/^[^@]+@[^@]+\\.[^@]+$/': { textContent: '‚úì Valid email format', style: { color: 'green' } }, '/^\\d+$/': { textContent: 'Numbers only', style: { color: 'blue' } }, '/hello/i': { textContent: 'Contains "hello" (case insensitive)', style: { color: 'purple' } } }, '#regex-output');

5. Empty, Null, and Undefined Checks

Special matchers for checking empty values, null, and undefined.
Select a value type
Conditions.apply(value, { 'empty': { textContent: 'Value is empty', style: { color: 'gray' } }, 'null': { textContent: 'Value is null', style: { color: 'orange' } }, 'undefined': { textContent: 'Value is undefined', style: { color: 'red' } }, 'truthy': { textContent: '‚úì Value exists', style: { color: 'green' } } }, '#special-output');
