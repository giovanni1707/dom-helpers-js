Storage Module

Enhanced localStorage and sessionStorage with automatic JSON serialization

Installation

<script src="https://cdn.jsdelivr.net/npm/@giovanni1707/dom-helpers@1.0.0/dist/dom-helpers-storage.min.js"></script>

API Methods

Storage.set(key, value)

Store any value with automatic JSON serialization

// Store objects, arrays, primitives
Storage.set('user', { name: 'John', age: 30 });
Storage.set('settings', { theme: 'dark', lang: 'en' });
Storage.set('count', 42);

Storage.get(key, defaultValue)

Retrieve values with automatic deserialization

const user = Storage.get('user');
const settings = Storage.get('settings', { theme: 'light' });
const count = Storage.get('count', 0);

Storage.remove(key) & Storage.clear()

Storage.remove('user');
Storage.clear(); // Clear all storage

Session Storage

// Use SessionStorage for temporary data
SessionStorage.set('tempData', { value: 123 });
const data = SessionStorage.get('tempData');
