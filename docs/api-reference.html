API Reference

Complete documentation for all DOM Helpers functions and methods

Core API

The core module provides three main helpers for DOM manipulation: Elements, Collections, and Selector.

View Full Core Documentation

Elements.{id}

Access and manipulate elements by their ID

Elements.myButton.update({
    textContent: 'Click Me',
    style: { backgroundColor: '#007bff' }
});

Collections.ClassName.{name}

Work with all elements sharing a class name

Collections.ClassName.btn.update({
    style: { padding: '10px 20px' }
});

Selector.query(selector)

Query elements using CSS selectors

Selector.query('.container').update({
    style: { padding: '20px' }
});

Storage API

Enhanced localStorage and sessionStorage with automatic JSON serialization.

View Full Storage Documentation

Storage.set(key, value)

Store data in localStorage with automatic serialization

Storage.set('user', { name: 'John', age: 30 });

Storage.get(key)

Retrieve data with automatic deserialization

const user = Storage.get('user');

Form API

Advanced form handling, validation, and data serialization.

View Full Form Documentation

Form.serialize(formElement)

Convert form data to JavaScript object

const data = Form.serialize(Elements.myForm);

Form.validate(formElement, rules)

Validate form fields with custom rules

const isValid = Form.validate(Elements.myForm, {
    email: 'required|email',
    password: 'required|min:8'
});

Animation API

Animate.fadeIn(element, duration)
