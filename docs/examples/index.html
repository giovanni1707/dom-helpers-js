Counter App

Basic element manipulation with the Elements helper

Code

let count = 0;

Elements.incrementBtn.update({
    addEventListener: {
        click: () => {
            count++;
            Elements.counterDisplay
                .update({
                    textContent: count
                });
        }
    }
});

Todo List

Dynamic list management with Collections

Code

Elements.addTodoBtn.update({
    addEventListener: {
        click: () => {
            const text = Elements.todoInput.value;
            if (!text) return;
            
            const li = document.createElement('li');
            li.textContent = text;
            li.className = 'todo-item';
            Elements.todoList.appendChild(li);
            
            Elements.todoInput.value = '';
        }
    }
});

    Theme Switcher

    Dynamic styling with the update method

    Code

    const themes = {
    blue: '#3b82f6',
    purple: '#8b5cf6',
    green: '#10b981',
    red: '#ef4444'
};

Collections.ClassName.themeBtn
    .forEach(btn => {
        btn.addEventListener('click', () => {
            const color = btn.dataset.theme;
            Elements.themeBox.update({
                style: {
                    backgroundColor: themes[color]
                }
            });
        });
    });

    Form Handling

    Capture and display form data

    Code

    Elements.demoForm.update({
    addEventListener: {
        submit: (e) => {
            e.preventDefault();
            const data = {
                name: Elements.userName.value,
                email: Elements.userEmail.value
            };
            Elements.formOutput.update({
                innerHTML: `
                    <strong>Name:</strong> ${data.name}<br>
                    <strong>Email:</strong> ${data.email}
                `
            });
        }
    }
});
